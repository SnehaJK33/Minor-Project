🌿 West Bengal Deforestation Analytics Dashboard

Visualizing forest loss trends and promoting environmental awareness through data-driven insights

🌲 About the Project

The Deforestation Analytics Dashboard is a data visualization project developed to analyze deforestation trends across West Bengal. It utilizes forest cover datasets to present insightful patterns and highlight environmental impacts caused by deforestation.

The system allows users to explore district-wise and state-level analytics through an interactive and visually appealing dashboard. This initiative aims to support research, awareness, and sustainable planning for a greener future.

🛠️ Tools & Technologies Used

  • Data Analysis: Pandas, NumPy
  • Machine Learning Model: Scikit-learn Linear Regression
  • Dataset: Forest cover data of West Bengal (Government & Satellite Sources)
